# Ideas Bucket

Autonomous product ideas for Enchantia Realms. Self-reviewed for profitability and feasibility.

**Rules:**
- Remove ideas when development starts
- Prioritize by: market size × pain intensity × simplicity
- Each idea needs: problem, target user, MVP scope, revenue model

---

## Status: Evaluating

### 1. Contract Clause Extractor
**Problem:** Small businesses sign contracts without understanding key clauses (termination, liability, auto-renewal)
**Target:** Freelancers, small agency owners
**MVP:** Upload PDF/Doc → highlights risky clauses with plain-English explanation
**Revenue:** $9/month or $49/document
**Stack:** Node.js + PDF parsing + OpenAI API

### 2. Calendar Ghost
**Problem:** People waste time finding meeting slots across multiple calendars (work + personal + side projects)
**Target:** Consultants, freelancers with fragmented schedules
**MVP:** Connect calendars → auto-suggests 3 best slots → booking link with buffer time
**Revenue:** $7/month personal, $19/month team
**Stack:** Node.js + Google/Outlook APIs + Calendly competitor

### 3. Invoice Intelligence
**Problem:** Freelancers lose money because they don't track which invoices are overdue or which clients always pay late
**Target:** Freelancers, small agencies
**MVP:** Email scan → auto-extracts invoices → tracks payment status → alerts on overdue
**Revenue:** $12/month
**Stack:** Node.js + Gmail API + Stripe for payments

### 4. Content Deja Vu
**Problem:** Content creators rewrite the same ideas for different platforms (Twitter thread → LinkedIn post → Newsletter)
**Target:** Solo creators, small marketing teams
**MVP:** Paste content → auto-rewrites for 3 platforms with format optimization
**Revenue:** $15/month
**Stack:** Node.js + OpenAI API + platform APIs

### 5. Policy Monitor
**Problem:** Businesses must comply with changing regulations (privacy policies, terms) but don't track updates
**Target:** E-commerce, SaaS, healthcare apps
**MVP:** Monitor competitor/policy sites → alerts on changes → suggests required updates
**Revenue:** $29/month (higher value, compliance-sensitive)
**Stack:** Node.js + web scraping + diff monitoring
